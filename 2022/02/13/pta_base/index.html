PTA基础题解

6-2 多项式求值 (15 分)

本题要求实现一个函数，计算阶数为n，系数为a[0] … a[n]的多项式f(x)=∑ (i=0~n)(a[i]*x^i) 在x点的值。

函数接口定义：

double f( int n, double a[], double x );

其中n是多项式的阶数，a[]中存储系数，x是给定点。函数须返回多项式f(x)的值。

裁判测试程序样例：

#include <stdio.h>

#define MAXN 10

double f( int n, double a[], double x );

int main()
&#123;
    int n, i;
    double a[MAXN], x;
    
    scanf("%d %lf", &n, &x);
    for ( i=0; i<=n; i++ )
        scanf("%lf", &a[i]);
    printf("%.1f\n", f(n, a, x));
    return 0;
&#125;

/* 你的代码将被嵌在这里 */

分析：

逻辑过程为：

  1. 获取每个数组元素；
  2. 通过获取的数组index计算并保存x的幂；
  3. 数组元素*x的幂；
  4. 上述相加，返回。

实现：

double f( int n, double a[], double x )
&#123;
    int i=1;
    double sum=a[0],m=1.00;
    for(i;i<=n;i++)//可以同时获取index与value；
    &#123;
        m*=x;//m储存x的幂；
        sum+=a[i]*m;
    &#125;
    return sum;
&#125;

思考

sum=0;从数组最开始获取如何？
答：不可以。m在i=0时也会*x，造成错误。
